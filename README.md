# WUA Studio — Capability Statement 2026

A single-page capability statement for WUA Studio.

## Local preview
Just open `index.html` in any browser, or run:
```bash
python3 -m http.server 8000
```
then visit http://localhost:8000

## Deploy
This repo can be deployed to Vercel, Netlify, or any static host.
Recommended: connect this repo to Vercel for automatic deploys on push.
